Nous migrons le DSL SWDiscovery vers une nouvelle API. Voici les décisions prises :

1. FERMETURES IMBRIQUÉES
   - something("x", x => x...) remplace le chaînage plat
   - from("x", x => x...) remplace focus("x")
   - Pattern : f(inner).copy(fn = this.fn) dans toutes les méthodes de traversée
   - focus() → @deprecated, conservé pour compatibilité

2. RENOMMAGE
   - SWDiscovery             → UnravelSession
   - SWTransaction           → UnravelQuery
   - SWDiscoveryConfiguration → UnravelConfig

3. PRIMITIVES DE TRAVERSÉE
   - out(p, ref) : ?focus <p> ?ref  (ancien isSubjectOf)
   - in(p, ref)  : ?ref <p> ?focus  (ancien isObjectOf)
   - p accepte URI | Var  et ref accepte URI | Var | Literal (orthogonalité totale)
   - isLinkTo / isLinkFrom → out(Var(...)) / in(Var(...))

4. ALIAS DOCUMENTÉS (sucre syntaxique sur out/in)
   - traverse(p, ref) = out(p,ref) + in(p,ref) mergés dans UnravelQuery (ref peut etre sujet ou objet de la relation propriete p)
   - isA(uri)         = out(URI("rdf:type"), uri)

5. REFONTE select → project (optionnel)
   - select() renommé project() : intention explicite = masquer les variables intermédiaires
   - project() est OPTIONNEL : sans lui, toutes les variables déclarées sont retournées
   - project() ignoré par asForm(), asRdf(), asNeo4j() qui veulent toutes les variables
   - select() conservé avec @deprecated pour rétrocompatibilité
   - Seul cas justifiant project() : nœud de traversal intermédiaire non voulu en sortie

6. UNRAVEL FORM (asForm())
   - Le bundle génère une IHM automatiquement depuis la définition de la requête
   - Chaque variable devient un composant UI selon type XSD + cardinalité :
       * datatype XSD numérique        → input numérique avec complétion
       * datatype XSD string           → input texte avec complétion
       * objet URI, < 20 valeurs       → listbox
       * objet URI, ≥ 20 valeurs       → input avec autocomplétion
   - Tableau résultats dynamique : tous les résultats si aucun filtre, filtré sinon
   - Paradigme : "Unravel Form" (SPARQL Forms piloté par UnravelSession)
